# HEAR YOUR HOME REST API

## Requirements

- Docker
- Docker-compose
- Node >= 10
- Npm

## Start the API

To edit the code in real time:

1) source .envrc
2) npm install
3) npm run db (Run a local database)
4) npm run redis
4) npm run dev

To run in production mode: 

1) source .envrc
2) docker-compose up --build
